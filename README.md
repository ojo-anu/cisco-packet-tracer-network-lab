# Enterprise Network Infrastructure Lab (Cisco Packet Tracer)

## Overview

This project demonstrates the design and configuration of a small enterprise network using Cisco Packet Tracer.

The lab simulates a real IT infrastructure environment including servers, routers, switches, wireless devices, and IoT integration.

The objective of this project was to practice network topology design, service deployment, and connectivity across multiple devices.

## Network Components

The network includes:

Router  
Two Switches  
Wireless Access Point  
DHCP Server  
Email Server  
FTP Server  
HTTP Server  
IoT Device  

## Network Architecture

The topology was designed with the following structure:

Router connected to a switch  
Switch-to-switch connection for network expansion  
Servers connected to the switch infrastructure  
Access point connected to a switch  
Wireless clients connected through the access point  
IoT devices integrated into the network

## Services Configured

DHCP Server  
Automatically assigns IP addresses to network devices.

HTTP Server  
Provides web service access within the network.

FTP Server  
Allows file transfer between devices.

Email Server  
Simulates internal email communication.

IoT Device Connectivity  
Demonstrates integration of smart devices in the network.

## Skills Demonstrated

Network Topology Design  
Router Configuration  
Switch Configuration  
DHCP Configuration  
Server Deployment  
Wireless Networking  
Enterprise Network Simulation

## Result

All devices were successfully connected within the network.  
Clients received IP addresses from the DHCP server, and services such as HTTP, FTP, and email were accessible across the network.
